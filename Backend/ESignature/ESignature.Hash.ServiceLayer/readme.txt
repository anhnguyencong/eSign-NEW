THƯ VIỆN VÀ MÃ NGUỒN MẪU
=========================

Thư viện RSSP.AgentSdk.CSharp.dll đã được đóng gói sẵn trong thư mục libs/. Khi sử dụng, bạn cần cài đặt thêm thư viện phụ thuộc sau:

    - Newtonsoft.Json (v13.0.3)

Gói NuGet RSSP.AgentSdk.CSharp.nupkg cũng được cung cấp tại thư mục libs/, cho phép bạn dễ dàng tích hợp SDK thông qua NuGet Package Manager trong Visual Studio hoặc các công cụ quản lý gói khác.

Thư viện được phát triển trên nền tảng .NET Standard 2.0, đảm bảo tương thích với:

    - .NET Framework 4.6.1 trở lên
    - .NET Core 2.0 trở lên
    - Các phiên bản .NET hiện đại: 5, 6, 7, 8 và 9

Mã nguồn mẫu (demo) minh họa cách sử dụng SDK được viết trên nền tảng .NET 8.
